# Collections – ArrayList

Bu klasörde Java'da **ArrayList** kullanımı
örnekler üzerinden gösterilmiştir.

## İçerik
- Generic olmayan ArrayList kullanımı
- Generic (ArrayList<Integer>) kullanımı
- for-each döngüsü ile liste üzerinde gezinme
- Sayısal değerlerin toplanması

## Amaç
Java Collections Framework yapısını,
ArrayList kullanımını ve
generic kullanımın sağladığı avantajları kavramak.
