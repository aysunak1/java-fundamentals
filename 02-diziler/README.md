# Diziler (Arrays)

Bu klasörde Java'da dizilerin kullanımı örneklenmiştir.

İçerik:
- String dizi tanımlama ve eleman atama
- for döngüsü ile dizi üzerinde gezinme
- for-each kullanımı
- Sayısal dizide toplam ve maksimum değer bulma

Amaç: Dizilerin temel kullanım mantığını ve pratik örneklerini öğrenmek.
