# Metot Overloading

Bu klasörde Java'da **metot overloading** kullanımı
örneklenmiştir.

## İçerik
- Aynı isimli metotların farklı parametrelerle tanımlanması
- Varargs kullanımı
- Yapıcı metot (constructor) kullanımı

## Amaç
Metot overloading mantığını ve
farklı parametre yapılarıyla metot kullanımını kavramak.
