# File Reading (Dosya Okuma)

Bu klasörde Java'da **dosya okuma** işlemi
BufferedReader kullanılarak örneklenmiştir.

## İçerik
- FileReader ve BufferedReader kullanımı
- Dosyadan satır satır veri okuma
- Okunan verilerin sayıya çevrilmesi
- try-catch-finally ile hata yönetimi
- finally bloğunda kaynak kapatma

## Amaç
Dosya okuma işlemlerini güvenli şekilde yapmayı,
hata yönetimini ve kaynakların doğru biçimde
kapatılmasını kavramak.
