# File Operations (Dosya İşlemleri)

Bu klasörde Java'da **dosya oluşturma, okuma, yazma ve dosya bilgisi alma**
işlemleri örneklenmiştir.

## İçerik
- File sınıfı ile dosya oluşturma
- Dosya bilgilerini alma (isim, yol, boyut, erişim izinleri)
- Scanner ile dosya okuma
- FileWriter ve BufferedWriter ile dosyaya yazma
- try-catch ile hata yönetimi

## Amaç
Java'da dosya işlemlerinin nasıl yapıldığını,
okuma–yazma süreçlerini ve
kaynak yönetimini kavramak.
