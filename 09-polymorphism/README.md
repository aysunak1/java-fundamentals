# Polymorphism (Çok Biçimlilik)

Bu klasörde Java'da **polymorphism** kullanımı
logger örneği üzerinden gösterilmiştir.

## İçerik
- Base sınıf (BaseLogger) tanımı
- Alt sınıflar ile metot ezme (override)
- Constructor üzerinden bağımlılık verme
- Aynı metot çağrısının farklı davranışlar sergilemesi

## Amaç
Polymorphism mantığını ve
üst sınıf referansı ile farklı nesnelerin
kullanılabilmesini kavramak.
