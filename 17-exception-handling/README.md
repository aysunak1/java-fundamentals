# Exception Handling (Hata Yönetimi)

Bu klasörde Java'da **exception handling** yapısı
try-catch-finally blokları üzerinden gösterilmiştir.

## İçerik
- try-catch kullanımı
- Farklı exception türlerinin yakalanması
- Genel Exception yakalama
- finally bloğunun kullanımı

## Amaç
Çalışma zamanında oluşabilecek hataları
kontrollü şekilde yakalamayı ve
programın akışını güvenli biçimde yönetmeyi kavramak.
