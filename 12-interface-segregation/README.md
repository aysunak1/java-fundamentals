# Interface Segregation

Bu klasörde Java'da **interface segregation** mantığı
çalışan örneği üzerinden gösterilmiştir.

## İçerik
- Farklı sorumluluklar için ayrı interface tanımları
- Bir sınıfın birden fazla interface’i implement etmesi
- İhtiyaca göre interface implementasyonu (OutSourceWorker)

## Amaç
Interface Segregation Principle (ISP) yaklaşımını ve
sınıfların yalnızca ihtiyaç duydukları arayüzleri
kullanmasını kavramak.
